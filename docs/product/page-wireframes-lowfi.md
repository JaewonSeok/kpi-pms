# KPI 성과관리 페이지별 Low-Fidelity Wireframes

이 문서는 [performance-management-prd.md](./performance-management-prd.md)를 바탕으로 각 핵심 페이지의 저충실도 텍스트 와이어프레임을 구현 가능한 수준까지 풀어쓴 문서다.

표기 규칙:
- `[]` 버튼 또는 입력
- `()` 보조 정보
- `---` 구획 분리
- `*` 중요 상태 또는 운영 규칙

## 공통 화면 규칙

모든 페이지 공통 헤더:

```text
[페이지명]                          [연도/주기 선택] [조직 범위] [도움말]
한 줄 설명
주요 상태 배너 / 마감일 / 공개일 / 권한 안내
```

공통 본문 규칙:

```text
[요약 지표 3~4개]
--------------------------------------------------
[필터 / 검색 / 탭 / 정렬]
--------------------------------------------------
[주 콘텐츠 영역]
--------------------------------------------------
[우측 상세 패널 또는 하단 이력 / 도움말]
```

모바일 공통 규칙:
- 상단에는 페이지명, 핵심 상태, 현재 선택 범위만 유지
- 대형 테이블은 카드 목록으로 축약
- 편집은 drawer 또는 full-screen form 우선
- 승인/제출 CTA는 하단 sticky action bar 사용

## 1. 조직 KPI `/kpi/org`

### Desktop Wireframe

```text
[조직 KPI]                                   [2026] [전사] [새 KPI 추가]
조직 목표를 정의하고 하위 조직 및 개인 KPI 연결 상태를 관리합니다.
(확정 전 KPI 12건 / 미연결 KPI 7건 / cascade율 68%)

[총 KPI 42] [확정률 71%] [미연결 7] [잠금 14]

--------------------------------------------------
[검색: KPI명/Owner] [상태] [조직레벨] [부서] [연결여부] [보기: 목표맵|목록|이력]
--------------------------------------------------
| 좌측: 조직 트리             | 중앙: KPI 목록 테이블                         | 우측: 상세 패널                 |
|----------------------------|----------------------------------------------|---------------------------------|
| 전사                       | KPI명 | 상위 KPI | 목표값 | 가중치 | 상태 | KPI명                            |
|  ㄴ 영업본부               | 매출 성장 | 전사 매출 | 15% | 20 | 확정 | 정의                              |
|  ㄴ 개발본부               | 원가 절감 | 수익성    | 5%  | 15 | 초안 | 산식 / 단위 / 난이도             |
|  ㄴ 경영지원               | ...                                        | Owner / 최근 변경자             |
|                            |                                              | 연결된 개인 KPI 14건            |
|                            |                                              | [개인 KPI 보기] [수정] [확정]   |
|                            |                                              | 변경 이력 타임라인               |
```

### Mobile Wireframe

```text
[조직 KPI]
2026 / 개발본부

[확정률 71%] [미연결 7]

[목표맵] [목록] [이력]
[부서 선택]
[상태 필터]

<KPI 카드>
KPI명 / 상태
목표값 / 가중치 / 연결 수
[상세 보기]

[+ KPI 추가]
```

### 핵심 인터랙션
- 조직 트리 클릭 시 KPI 목록 즉시 동기화
- KPI 선택 시 우측 패널에서 상세/연결/이력 확인
- 확정 버튼 클릭 시 하위 개인 KPI 영향 요약 모달 노출
- 목표맵 보기에서는 상위 KPI와 하위 부서 KPI 연결선을 시각적으로 표시

### 상태/예외
- Empty: "아직 조직 KPI가 없습니다" + `첫 KPI 추가`
- Lock 상태에서는 수정 대신 `잠금 해제 요청`
- 개인 KPI 연결이 하나도 없는 확정 KPI는 경고 배지 표시

- 잠금 후에는 읽기 전용 + 수정 요청 버튼

- 누적 추이 탭에서 월별 달성률 mini chart 표시

### 상태/예외
- 해당 월이 마감되면 수정 차단 + 마감 안내
- 개인 KPI가 없으면 월간 실적 입력 대신 KPI 작성 유도
