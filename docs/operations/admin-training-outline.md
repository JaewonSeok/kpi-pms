# Admin Training Outline

## Audience

- HR admins
- division heads involved in evaluation/compensation review
- launch support coordinators

## Session structure

### 1. System overview

- login and role model
- main navigation
- what is different from legacy system

### 2. Organization and user administration

- org chart upload preview vs apply
- common validation errors
- post-import reconciliation checks

### 3. KPI and evaluation operations

- KPI setup and confirmation
- evaluation workflow basics
- AI assist approval flow
- result publication checkpoints

### 4. Compensation and notification operations

- rule set / scenario basics
- approval workflow states
- notification dead-letter and retry review

### 5. Operations dashboard and runbooks

- `/admin/ops` walkthrough
- health checks
- feature flags
- when to escalate and where to look first

## Practice labs

1. preview org upload
2. identify and fix a bad department code
3. approve an AI draft
4. review dead-letter count and job execution
5. follow go-live checklist and incident checklist
